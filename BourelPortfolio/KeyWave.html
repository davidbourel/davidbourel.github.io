Key Wave

Global Game Jam 2017

KeyWave est un coureur musical dans lequel le joueur peut jouer du piano pour modifier la vague sur laquelle il surfe afin d'éviter les obstacles et les ennemis.

Développé sur Unity lors du Global Game Jam 2017 en 48h.
